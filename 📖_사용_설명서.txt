=========================================
📚 LMS 프로젝트 관리 도구 사용법
=========================================

이제 4개의 도구를 사용할 수 있습니다:

1️⃣ LMS_개발_가이드.docx
   - 전체적인 개발 전략과 방법론
   - 문제 해결 방법
   - 체계적인 개발 프로세스

2️⃣ 프로젝트_현황_템플릿.txt
   - 프로젝트 상태를 기록하는 문서
   - 매일 업데이트하세요
   - Claude와 대화할 때 참고 자료로 사용

3️⃣ Claude_대화_템플릿.txt
   - Claude와 효율적으로 대화하는 템플릿
   - 복사-붙여넣기해서 사용
   - 상황별 템플릿 제공

4️⃣ 프로젝트_백업하기.bat
   - 더블클릭으로 간단히 백업
   - 자동으로 날짜/시간 폴더 생성
   - 메모 기능 포함

5️⃣ 프로젝트_복원하기.bat
   - 문제 발생 시 이전 버전으로 복원
   - 백업 목록에서 선택
   - 안전 확인 절차 포함

=========================================
🚀 빠른 시작 가이드
=========================================

STEP 1: 준비하기
1. "프로젝트_현황_템플릿.txt"를 복사
2. "프로젝트_현황.txt"로 이름 변경
3. 현재 프로젝트 상태 작성

STEP 2: 백업하기
1. "프로젝트_백업하기.bat" 더블클릭
2. 메모 입력 (예: "출석 기능 추가 전")

STEP 3: Claude와 작업하기
1. "Claude_대화_템플릿.txt" 열기
2. 해당하는 템플릿 복사
3. [ ] 부분을 실제 정보로 수정
4. Claude에게 전송

STEP 4: 작업 후 기록
1. "프로젝트_현황.txt" 업데이트
2. 완성된 기능 체크
3. 개발 로그 작성

=========================================
⚠️ 주의사항
=========================================

• 새 기능 추가 전에는 반드시 백업
• 한 번에 하나의 기능만 개발
• 문제 발생 시 즉시 복원
• 매일 프로젝트 현황 업데이트

=========================================
💡 프로 팁
=========================================

1. 폴더 구조 정리
   backups/         ← 백업 파일들
   src/
     ├─ student/    ← 학생 관리
     ├─ teacher/    ← 강사 관리
     ├─ attendance/ ← 출석 관리
     └─ grade/      ← 성적 관리

2. Claude 대화 관리
   - 대화가 20회 이상 길어지면 새 창
   - 기능별로 대화 분리
   - 명확한 범위 지정

3. 테스트 체크리스트
   ✓ 로그인 기능
   ✓ 학생 목록
   ✓ 페이지 이동
   ✓ 에러 없음

=========================================
📞 도움이 필요하신가요?
=========================================

문제가 해결되지 않으면:
1. 백업에서 복원
2. 더 작은 단위로 나누어 시도
3. Claude에게 구체적으로 질문

화이팅! 프로젝트 성공을 응원합니다! 🎯
